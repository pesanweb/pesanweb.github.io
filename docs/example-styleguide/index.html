Example components

Components used for documented Single File Component features

example_aliases

This component documents how aliases can be used on your site.

Available context

name
The name to say hello to.

Example usage

Basic usage
Say hello to a user by providing their name.
{% include "sfc--example-aliases" with {"name": "Sam"} %}
Hello Sam!

example_styleguide_component

This component allows components to be documented in the styleguide. So meta.

Available context

title
The component title or ID
description
A description for the component.
context
An associative array mapping context names to descriptions.
usage
An array of associative arrays that document example usages.
usage[].title
The title of the example.
usage[].description
A description for the example.
usage[].twig
The Twig snippet for the example.

Example usage

Basic usage
Display information for a hypothetical component.
{% include "sfc--example-styleguide-component" with { "title": "my_component", "description": "A description for the component", "context": { "variable": "A context variable", }, "usage": [ { "title": "Basic usage", "description": "Some description for this example", "twig": "{% include 'sfc--example-aliases' with {'name': 'Sam'} %}", }, ], } %}

my_component

A description for the component

Available context

variable
A context variable

Example usage

Basic usage
Some description for this example
{% include 'sfc--example-aliases' with {'name': 'Sam'} %}
